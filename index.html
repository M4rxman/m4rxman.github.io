Про проєкт

Розробка повнофункціональної Top-Down Shooter гри з використанням модульної архітектури, інноваційного PID контролера для штучного інтелекту та системи тестування на платформі Godot Engine

Актуальність дослідження

🎮

Зростання ігрової індустрії

Розвиток ігрової індустрії та зростання популярності незалежної розробки створює потребу в дослідженні можливостей сучасних безкоштовних ігрових рушіїв.

⚡

Godot Engine

Як один з найперспективніших opensource рушіїв, Godot потребує детального вивчення можливостей для створення складних ігрових систем.

🏗️

Архітектурні рішення

Потреба в дослідженні ефективних архітектурних підходів до створення модульних та масштабованих ігрових систем.

Мета дослідження

Розробка інформаційної ігрової системи на рушії Godot в жанрі шутер з реалізацією повного циклу ігрових механік, модульної архітектури та комплексної системи тестування.

Архітектура системи

Модульний архітектурний підхід з використанням принципів SOLID та Clean Architecture

Presentation Layer

  • Main_scene.tscn - головна сцена гри
  • UI компоненти
  • Система введення

Business Logic Layer

  • GameManager.gd - центральний менеджер
  • Player контролер
  • Enemy AI система
  • Weapon система

Data Layer

  • SaveManager.gd - система збереження
  • Конфігураційні файли
  • Ресурси гри

Основні завдання

  • Аналіз предметної області - дослідження сучасного стану розробки ігор на Godot Engine та особливості жанру шутер
  • Архітектурні підходи - дослідження та розробка модульних ігрових систем з чітким розділенням відповідальностей
  • Система управління персонажем - розробка з реалістичною фізикою руху на основі RigidBody3D
  • Штучний інтелект - створення системи ШІ для ворогів з адаптивною поведінкою та PID контролером
  • Бойові механіки - реалізація системи зброї та бойових взаємодій
  • Система збереження - розробка надійної системи збереження та завантаження з JSON серіалізацією
  • Тестування - створення комплексної системи тестування з використанням GUT фреймворку

Інноваційні рішення

🧠

PID Контролер для ШІ

Використання промислових алгоритмів управління для ігрових застосувань. Коефіцієнти: Kp=25.0, Ki=0.1, Kd=1.0 забезпечують оптимальну поведінку ворогів.

⚙️

RigidBody3D Фізика

Заміна CharacterBody3D на RigidBody3D для всіх персонажів забезпечила природну фізичну взаємодію та реалістичну поведінку.

💾

JSON Серіалізація

Гнучка система збереження з використанням JSON формату, що забезпечує читабельність та розширюваність даних.

🧪

Автоматизоване тестування

Комплексна система тестування з використанням GUT фреймворку для забезпечення якості та надійності коду.
